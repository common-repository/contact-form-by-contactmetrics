== Description ==
This plugin allows you to create a simple, elegant, contact form in minutes.

= Free Features =

* AJAX form validation
* SPAM detection (no CAPTCHA required)
* Customizable CSS

This plugin can be used a simple stand-alone form, or as a means of embedding an advanced form customized at http://contactmetrics.com.  When used to it's full capabilities, it includes the following features.

= Upgradable Features =

* Drag-and-drop form builder 
* Detailed analytics about each visitor who contact you
* Multiple email recipients
* Much more

== Installation ==

1. Upload 'contact-form-contactmetrics' to the '/wp-content/plugins/' directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Click on Contact Form in your WordPress Menu
4.  Follow the steps to upgrade your map at ContactMetrics.com
5. Paste the shortcode [contactmetrics] into your post or page where you want the contact form to appear.
